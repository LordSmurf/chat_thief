# Simple Python Twitch Chat Client with no dependencies

Set the following environment variables:

- TWITCH_OAUTH_TOKEN
- TWITCH_CHANNEL
- TWITCH_BOT_NAME

```bash
python bot.py
```

## Goals

- I want chat_thief to be unix-like:
  - Send messages to chat
  - Collect messages and send them somewhere

```bash
beginbot

beginchat
```
